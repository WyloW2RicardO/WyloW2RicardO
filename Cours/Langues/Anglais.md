### Adverb

#### Frequency

L'advebe de fréquance se place toujours ***devant*** le verbe sauf avec etre [be].

| --------- | -------- |
| always    | toujours |

| sometimes | parfois  |
| rarely    | rarement |
| never     | jamais   |
